To get started:

1. Set up python virtual environment:
    - python -m venv env
    - source env/bin/activate
2. Install Utilities
    - brew install sqlite3 curl
3. Install Libraries
    - pip install reqs.txt
    - pip install -e .
4. Reset DB
    - ./bin/db resetHard
5. Run app
    - ./bin/run
    - Navigate to localhost:8000
